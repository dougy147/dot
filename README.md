# dot

`dot` déploie automatiquement un environnement graphique pour des distributions Linux basées sur Arch.

## Installation

Sur une distribution Arch fraîchement installée :

```
curl -LO dougy147.com/dot/dot.sh
sh ./dot.sh
```
